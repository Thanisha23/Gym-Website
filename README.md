## Project Setup

1. **Clone the repository**
```
git clone https://github.com/Thanisha23/Gym-Website.git
```

2. **Navigate to the project directory and install dependencies for the project**

 ```
pnpm install
```
3. **Start the frontend development server**
```
npm run start
```
